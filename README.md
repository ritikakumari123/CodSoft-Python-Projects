# CodSoft Python Internship Tasks

This repository contains Python projects completed as part of the CodSoft Internship Program.

## Projects Included

### Task 1 — To-Do List Application
A simple Python application to manage daily tasks efficiently.
Features:
- Add tasks
- View tasks
- Remove tasks

### Task 2 — Calculator
A basic calculator using Python that performs:
- Addition
- Subtraction
- Multiplication
- Division

### Task 3 — Password Generator
A Python program that generates strong and random passwords based on user-defined length.

### Task 4 — Rock Paper Scissors Game
A fun command-line game where the user plays against the computer.
Features:
- Random computer choice
- Winner detection
- Score tracking
- Multiple rounds

### Task 5 — Contact Book
A contact management system using Python.
Features:
- Add contact
- View contacts
- Search contact
- Update contact
- Delete contact

## Technologies Used

- Python
